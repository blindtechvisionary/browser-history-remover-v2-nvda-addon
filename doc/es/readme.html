Browser History Remover V2 - Documentación Completa

Descripción general

Browser History Remover V2 es un potente complemento de NVDA diseñado para ayudarte a gestionar y eliminar el historial de navegación de múltiples navegadores web desde una interfaz única y accesible. En lugar de abrir cada navegador individualmente y navegar por los menús de configuración, este complemento te permite manejar todo directamente desde NVDA.

La versión 2 introduce mejoras significativas, incluyendo un sistema de configuración que recuerda tus preferencias, la capacidad de realizar copias de seguridad del historial antes de su eliminación y una función de eliminación rápida que te permite borrar el historial de tu navegador predeterminado con un solo atajo de teclado.

Una capacidad importante de este complemento es su habilidad para borrar los datos del historial local incluso cuando tienes la sesión iniciada en tu cuenta del navegador. Ya sea que uses Chrome Sync, Firefox Sync o servicios similares, esta herramienta elimina eficazmente los rastros locales de tu actividad de navegación.

Novedades de la Versión 2

  • Diálogo de configuración: Un panel de ajustes dedicado donde puedes personalizar el comportamiento del complemento.
  • Opción de respaldo del historial: Elige guardar una copia de tu historial antes de borrarlo, proporcionando una red de seguridad en caso de que necesites recuperar algo más tarde.
  • Selección de navegador predeterminado: Establece tu navegador preferido para las operaciones de eliminación rápida.
  • Atajo de eliminación rápida: Presiona NVDA+Alt+D para borrar instantáneamente el historial de tu navegador predeterminado sin abrir ningún diálogo.
  • Ajustes persistentes: Tus preferencias se guardan automáticamente y se recuerdan entre sesiones de NVDA.
  • Accesibilidad mejorada: Mejor navegación por teclado en todos los diálogos.

Navegadores Soportados

El complemento admite la eliminación del historial en los siguientes navegadores:

Navegadores basados en Chromium

  • Google Chrome
  • Microsoft Edge
  • Opera
  • Brave
  • Vivaldi
  • Chromium

Navegadores basados en Firefox

  • Mozilla Firefox
  • Waterfox
  • Pale Moon
  • Basilisk

Otros navegadores

  • SeaMonkey

Importante: Este complemento elimina el historial de navegación local almacenado en tu ordenador. No elimina el historial que se ha sincronizado con servicios en la nube como la Cuenta de Google o Firefox Sync. Para limpiar completamente tu huella de navegación, también debes gestionar los ajustes de sincronización dentro del propio navegador.

Primeros Pasos

Abrir el diálogo principal

Hay dos formas de abrir Browser History Remover:

  • Presiona NVDA+Alt+B desde cualquier lugar en Windows.
  • Navega al Menú NVDA, luego Herramientas y después Browser History Remover.

El Diálogo Principal

Al abrir el complemento, encontrarás una interfaz sencilla:

  • Lista de navegadores: Un cuadro de lista con todos los navegadores soportados. Usa las flechas para seleccionar el navegador cuyo historial deseas borrar.
  • Botón Eliminar historial (Alt+D): Elimina el historial del navegador seleccionado.
  • Botón Configuraciones (Alt+C): Abre el diálogo de configuración.
  • Botón Salir: Cierra el diálogo.

Opciones de Configuración

Presiona Alt+C desde el diálogo principal para acceder al panel de ajustes.

Copiar historial del navegador antes de la eliminación

Cuando esta casilla está activada, el complemento crea un respaldo de tus archivos de historial antes de borrarlos. El respaldo se guarda en:

C:\Users\TuUsuario\Downloads\browser_history_remover\history\NombreDelNavegador\

Navegador predeterminado para eliminación rápida

Selecciona tu navegador preferido de la lista desplegable. Este será el que se use al presionar el atajo de eliminación rápida (NVDA+Alt+D).

Botones de Guardado

  • Botón Guardar (Alt+S): Guarda tu configuración y vuelve al diálogo principal.
  • Botón Cancelar (Alt+C): Descarta los cambios y vuelve al diálogo principal.

Atajos de Teclado

Atajos Globales

Atajo Acción
NVDA+Alt+B Abrir el diálogo de Browser History Remover
NVDA+Alt+D Eliminación rápida del historial para el navegador predeterminado
